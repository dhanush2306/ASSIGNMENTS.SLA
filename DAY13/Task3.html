Understanding em vs rem Units

Parent Font Size = 20px

1.5em Text → 1.5 × 20px = 30px

1.5rem Text → 1.5 × 16px = 24px

Explanation

em: Scales based on the parent element's font size.

rem: Scales based on the root element (html) font size.

Resize or adjust the html or parent font size to see the difference.
